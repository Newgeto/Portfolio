# Portfolio — Yanis Mdoughy

Portfolio personnel développé avec React + Vite.

## Stack

- React 19
- Vite 7
- CSS vanilla (sans framework)

## Lancer le projet

```bash
npm install
npm run dev
```

## Scripts

| Commande | Description |
|----------|-------------|
| `npm run dev` | Serveur de développement |
| `npm run build` | Build de production |
| `npm run preview` | Prévisualiser le build |
| `npm run lint` | Linter ESLint |

## Structure

```
src/
├── components/   # Header, Footer
├── sections/     # About, Contact, Experience, Formation, Home, Projects, Skills
├── styles/       # CSS par composant
├── data/         # Données statiques (skills)
└── assets/       # Images, CV, SVG
```
